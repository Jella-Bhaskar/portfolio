# Jella Bhaskar — Professional Engineering Portfolio

Welcome to the source repository for my personal engineering portfolio website. Designed away from generic templates, this site embodies a **"Modern Intelligent Workspace"** theme—built specifically to project enterprise-grade engineering primitives, clean visual hierarchy, and production readiness.

## 🎨 Design Philosophy & Choices

- **The Intelligent Workspace Aesthetic:** Employs a pristine, high-contrast light canvas (`#fcfdfe`) balanced by crisp charcoal typography (`#1e293b`) and neon electric blue branding accents (`#0ea8ea`). This mirrors the exact design systems used by elite modern tech ecosystems like Stripe, Linear, and OpenAI.
- **Cursive Signature Branding:** Incorporates an elegant, handwritten logo fluidly rendered using the *Mea Culpa* script font, providing a highly personalized visual identity at the top of the viewport.
- **Glassmorphism Island Dock:** Replaces heavy, static navigation blocks with a translucent horizontal navigation island that floats natively on scroll. Navigation tabs feature fluid, sideways expansion micro-interactions upon user engagement.
- **Modular Data Blocks:** Information is grouped into clean, borderless grids and layout structures that display high density without clutter, making it effortless for engineering managers and recruiters to parse text quickly.

## 🚀 Specialized Core Frameworks Highlighted

- **Generative AI & LLMs:** Production pipeline architectures, Azure OpenAI (GPT-5), LangChain orchestration, RAG architectures, and agentic workflows.
- **Document AI & IDP:** Intelligent Document Processing pipelines, Azure Document Intelligence Studio, custom classification/extraction models, OCR parsing matrices, and FAISS vector retrieval systems.
- **Cloud Infrastructure & MLOps:** Containerization and clustering via Docker and Kubernetes (AKS/EKS), asynchronous queue routing, and automated CI/CD engine delivery.
